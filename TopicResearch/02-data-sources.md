# 数据源清单

## 核心数据源

### ifind 金融数据终端
- **用途**：LME 铜铝价格、上证指数、有色金属指数、M7 财务数据
- **调用方式**：`/home/ll/.agents/skills/ifind/call-node.js`
- **数据周期**：日级别（2005-01-04 ~ 2008-12-31）

### 数据文件

| 文件 | 记录数 | 来源指标 |
|------|--------|----------|
| `lme_copper_daily_2005_2008.json` | 1010 | LME 铜现货结算价 |
| `lme_aluminum_daily_2005_2008.json` | 1010 | LME 铝现货结算价 |
| `shcomp_daily_2005_2008.json` | 971 | 上证综指日收盘价 |
| `nonferrous_daily_2005_2008.json` | 971 | 申万有色金属指数日收盘价 |

### 其他数据源
- **SEC EDGAR**：M7 资本开支/营收数据
- **FRED**：M2、贸易顺差等宏观数据

## 引用规范
- 数据来源标注在图表注释中
- 阈值状态标注"初值/待校准"
- 时间戳使用 float 非 ISO 字符串
