================================================================================
 完整 Chat 流程图 — llmwikify
 生成时间: 2026-06-22
 状态: Phase 7 后 (provider injection + DreamScheduler lifespan)
================================================================================

核心特征:
  ◆ 3 个 HTTP 入口 (SSE / WebSocket / OpenAI-compat) → 1 个统一入口 (AgentService.chat)
  ◆ 1 个 ReAct 循环 (ChatRunnerV2.run_stream) → 1 个 LLM 调用 (astream_chat)
  ◆ 13 个 hook 点 (CompositeHook) 贯穿 5 步状态机
  ◆ 微压缩 (microcompact) 默认 ON: 大工具结果 → marker
