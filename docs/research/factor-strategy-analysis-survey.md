## 一、因子分析（Factor Analysis）

### 核心模块（P0 — 必选）

| 模块 | 图表类型 | 数据字段 |
|---|---|---|
| **IC/IR 统计** | IC 时间序列图 + IC 分布直方图 + 分年度 IC 统计表 | `date, rank_ic, ic_rolling_mean, ic_mean, ic_std, t_stat` |
| **分层回测** | 分层净值曲线（G1-G5）+ 多空净值曲线 + 分层收益柱状图 | `group, cumulative_return, annualized_return, sharpe` |
| **因子收益** | 因子收益时序图 + T 统计量柱状图 | `factor_return, t_stat, p_value` |
| **换手率** | 换手率时序图 + 成本拖累柱状图（毛收益 vs 净收益） | `turnover_rate, gross_return, net_return` |

### 重要模块（P1 — 推荐）

| 模块 | 图表类型 | 数据字段 |
|---|---|---|
| **因子衰减** | IC 衰减曲线（持有期 1-12 月） | `holding_period, ic_mean, icir` |
| **因子拥挤度** | 持仓集中度 + 容量估算 | `concentration_ratio, estimated_capacity` |
| **因子正交性** | 因子相关性矩阵热力图 | `factor_a, factor_b, correlation` |
| **市场状态检验** | 分状态 IC 柱状图（牛/熊/震荡） | `market_regime, ic_mean, icir` |

### 辅助模块（P2 — 可选）

| 模块 | 图表类型 |
|---|---|
| 行业/市值中性化效果 | 中性化前后 IC 对比柱状图 |
| 参数敏感性 | 参数网格热力图 |
| 因子分布统计 | 截面分布箱线图 |

## 二、策略分析（Strategy Analysis）

### 核心模块（P0 — 必选）

| 模块 | 图表类型 | 数据字段 |
|---|---|---|
| **KPI 概览** | 6-10 张指标卡片 | `cagr, sharpe, sortino, max_drawdown, win_rate, total_trades, alpha, beta` |
| **净值曲线** | 折线图（策略 vs 基准） | `date, equity, benchmark_equity` |
| **回撤图** | 水下曲线 + Top5 回撤标注 | `date, drawdown_pct, peak, trough` |

### 重要模块（P1 — 推荐）

| 模块 | 图表类型 | 数据字段 |
|---|---|---|
| **月度收益热力图** | 月份 × 年份矩阵（绿正红负） | `year, month, monthly_return_pct` |
| **年度收益** | 柱状图 + 均值虚线 | `year, annual_return_pct` |
| **风险指标** | VaR/CVaR/波动率数值卡 | `var_95, cvar_95, annual_volatility` |
| **策略 vs 基准对比** | 对比表格 + 超额收益曲线 | `excess_return, tracking_error, up_capture, down_capture` |

### 辅助模块（P2 — 可选）

| 模块 | 图表类型 |
|---|---|
| 持仓分布 | 饼图（时间加权平均持仓） |
| 交易统计 | 表格（盈亏比/最大连续盈亏/MAE/MFE） |
| 绩效归因 | Brinson 归因图 + 因子归因柱状图 |
| 压力测试 | 多个小倍图（危机事件 vs 基准） |
| 滚动统计 | 滚动 Sharpe/Beta/波动率双线图 |

## 三、与现有代码的差距

### 已有能力

| 模块 | 现有代码 |
|---|---|
| 基础指标 | `metrics.py`: sharpe, max_dd, win_rate, total_return（4 个） |
| 简单 PnL 图 | `backtest.py`: trades 列表（含 side, price, fee） |
| Wiki 页面写入 | `run.py`: BacktestResult + Optimization markdown |

### 缺失能力

| 模块 | 缺失内容 |
|---|---|
| **因子分析** | 完全缺失：无 IC 计算、无分层回测、无因子收益分析 |

| **收益分解** | 缺月度/年度收益热力图 |
| **回撤分析** | 缺水下曲线、Top5 回撤标注、回撤修复天数 |
| **交易统计** | 缺盈亏比/最大连续盈亏/MAE/MFE |
| **绩效归因** | 完全缺失 |
| **基准对比** | 缺超额收益曲线、捕获比率 |
| **压力测试** | 完全缺失 |

## 五、关键设计原则

1. **一图一结论**：每个图表传达明确信息（如"IC 显著为正"、"单调性良好"）
2. **统计检验**：所有 IC/收益必须附 T 统计量和 P 值
3. **成本敏感**：所有收益分析区分毛收益和净收益，标注换手率
4. **可折叠**：P1/P2 模块默认折叠，用户按需展开
5. **自包含**：每个图表标注时间范围、基准信息、关键事件
